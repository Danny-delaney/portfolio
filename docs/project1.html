Overview

What Is It?

A real-time networked platformer where an animated stickman character runs, jumps and punches across your actual desktop — using every open application window as a platform. The game state lives on a sender machine and streams compressed video to a receiver over TCP, which can be on the same PC or across a local network.

The receiver sends WASD / Space control inputs back, letting you play the game remotely while watching a live compressed stream of the sender's screen.

Features

What I Built

  • Custom TCP binary protocol streaming JPEG keyframes and delta patches at up to 18 FPS
  • Frame-differencing algorithm that only sends changed screen regions, reducing bandwidth by up to 80%
  • Transparent PyQt5 overlay rendered fullscreen above all other windows with click-through mouse events
  • Win32 window enumeration to detect all visible application windows and use them as collidable platforms
  • Full sprite animation system with state machine: idle, run, jump up/down, punch — all sprite-sheet sliced
  • Physics engine with gravity, wall jumping, and collision detection against dynamic window positions
  • "Enter window" mechanic — character can descend into a window and interact with its content
  • Left-click injection into Notepad via Win32 messages when the character punches inside it
  • YAML-driven level layout manager that repositions real application windows to form designed levels
  • Dot/collectible system with score counter rendered live on the overlay
