Overview

What Is It?

A computer vision pipeline that gives a DJI Tello drone two ways to be controlled: hand gesture recognition (thumbs up = takeoff, thumbs down = land, open palm = track hand) and autonomous calculator tracking using a custom YOLOv8 model trained from scratch.

A five-state mode machine handles seamless switching between modes — if a gesture is lost, the drone searches for a calculator target and vice versa, avoiding dead stops mid-flight. Includes a full debug mode with a laptop webcam so it can be developed without a drone.

Features

What Was Built

  • Trained a custom YOLOv8n model on a self-collected dataset for calculator detection
  • Integrated MediaPipe Gesture Recognizer running in VIDEO mode for low-latency hand tracking
  • Five-state mode machine that gracefully switches between gesture and calculator tracking targets
  • PD-style tracking: maps target offset from frame centre to drone RC velocity commands
  • Deadzone system to prevent jitter when target is already centred
  • Hue-shift colour correction to compensate for the Tello camera's warm colour cast
  • Cooldown system prevents repeated command spam to the drone
  • Full debug mode using laptop webcam — all logic runs without drone hardware connected
  • Hand skeleton drawn live on frame with all 21 MediaPipe landmarks and connections
